And's Homme

And's Hommes est un projet de site E-commerce pour des produits de cosmétiques pour hommes. Ce projet a été réalisé avec 2 autres personnes dans le cadre du projet de fin de premier semestre de notre deuxième année. Ce projet avait pour but de tester notre créativité en plus de vérifier nos compétences en front/back-end. Nous avons donc dû concevoir et planifier tout le projet de A à Z, sans consignes précises. Les seules contraintes étaient l'utilisation d'une base de données SQL et d'un framework PHP.

Pikomino

Dans le cadre de l'IUT nous avons réalisé par groupe de 4, un jeu en Kotlin et JavaFX basé sur le jeu de société Pikomino. L'objectif était d'imaginer comment fonctionnerait le jeu s'il était adapté en tant que jeu mobile.
